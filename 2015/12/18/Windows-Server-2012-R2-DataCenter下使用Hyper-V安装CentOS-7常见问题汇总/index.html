Windows Server 2012 R2 DataCenter下使用Hyper-V安装CentOS 7常见问题汇总

发表于 2015-12-18   |   分类于 Hyper-v   |  


环境说明

Windows操作系统版本：Windows Server 2012 R2 DataCenter
CentOS版本：CentOS-7-x86_64-DVD-1511.iso
Hyper-V版本：6.3.9600.16384

问题一：Failed Secure Boot Verification.

问题描述

开机后会出现如下提示：

原因及解决办法

这个主要是由于开启了Enable Secure Boot选项导致的，直接关掉该选项就可以了。

问题二：首次安装完成启动后，会有一个License Information，那里操作可能会有点歧义。

这里先按1，然后回车；再按2，然后回车，表示接受许可；然后再按q，然后回车，表示要退出；最后再输入yes，就可以了。此时系统就可以启动了。
