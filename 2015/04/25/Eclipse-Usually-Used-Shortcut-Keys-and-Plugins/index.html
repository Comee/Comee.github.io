Eclipse 常用快捷键与插件汇总

发表于 2015-04-25   |   分类于 IDE   |  

快捷键

Eclipse 中包含很多快捷键，记忆起来实在是比较困难。顾整理一些常用的快捷键，方便自己记忆。如果想要查看 Eclipse 中所有的快捷键，可以在 Eclipse 界面中，按 Ctrl+Shift+L 来查看详细的快捷键列表。此外，也可以设置自定义快捷键（按 2 次 Ctrl+Shift+L 可以打开 Eclipse 快捷键设置页面）：Window > Preferences > General > Keys。

在 Eclipse 中存活

其实，在 Eclipse 中即使不使用快捷键也可以存活，但是，基于现实中几乎没有人不用快捷键，所以下面的快捷键可以认为是存活级别的：

快捷键 说明
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + X 剪切
Ctrl + D 删除一行
Ctrl + Shift + F 格式化整个文件
Ctrl + N 新建对话框（新建文件、项目等）
F2 文件重命名
Ctrl + W 关闭 tab 页
Ctrl + S 保存
Ctrl + A 全选

在 Eclipse 中生活

想在 eclipse 中生活，就得再掌握几个常用的快捷键，这些快捷键可以加快编码速度：

快捷键 说明
Ctrl + F 查找替换
Ctrl + Shift + Enter 在当前行之前插入一行（不从光标后切断）
Shift + Enter 在当前行之后插入一行（不从光标后切断）
Ctrl + / 注释当前行（行注释）
Ctrl + Shift + / 注释整个选择区域（段落注释）
Ctrl + Shift + Y To Lower Case
Ctrl + Shift + X To Upper Case
Ctrl + Z 撤销（undo）
Ctrl + Y 重做（redo）
Ctrl + K Find Next (比 Ctrl+F 中的查找更好用)
Ctrl + Shift + K Find Previous (比 Ctrl+F 中的查找更好用)
Ctrl + Q 回到上一次编辑的地方
Ctrl + 2,R 批量重命名（Rename infile）
Alt + Shift + R 批量重命名（Rename infile）
Ctrl + F11 Run

在 Eclipse 中乐活

想在 eclipse 中乐活，就得再掌握几个不太常用的快捷键，这些快捷键可以加快操作速度：

快捷键 说明
Ctrl + Shift + R 打开资源（包括 jar 文件等）搜索框
Ctrl + Shift + T 打开类搜索框
Ctrl + Shift + O 自动导入必要的包（Package）
Ctrl + M 将 tab 页在最大化和适中之间切换
Ctrl + O 查看类的 Outline
Ctrl + E 在打开的多个 tab 页进行切换
Ctrl + . 定位到下一处错误
Ctrl + 1 建议，提示 （一般和 Ctrl+. 一起使用）
Ctrl + Alt + / Word Completion (类似 Unix 命令行下的 Tab )
Ctrl + I 缩进当前行（Correct indentation 当编辑别人的 SVN 代码的时候，
为了不打乱别人的格式，
这时候使用 Ctrl+Alt+I 比 Ctrl+Shift+F 更合适）
Ctrl + Alt + J Join Lines
Ctrl + Alt + G Find Selected Text in Workspace
Ctrl + F12 Open Task
Ctrl + 2,F Assign to field
Ctrl + 2,L Assign to local variable
Ctrl + 2,M Extract method
Alt + Shift + Q,C Show View (Console)
